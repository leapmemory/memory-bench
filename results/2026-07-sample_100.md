# Run: public sample corpus (July 2026)

Anyone can reproduce this run. Corpus, probes, harness, and grading are all in
this repository.

## Result

| Metric | Value |
|---|---|
| Probes | 12 (incl. one abstention probe) |
| Accuracy, pasted history | 12/12 |
| Accuracy, LeapMemory recall | 12/12 |
| Scored (both correct) | 12 |
| Median input-token reduction | 91.7% |
| Range | 86.6% - 97.4% |
| Model, both conditions | claude-sonnet-4.6 via OpenRouter, temperature 0 |
| Token source | provider-billed prompt_tokens |

## Corpus

`corpus/sample_100.jsonl`: a synthetic customer-support conversation, 98 turns
across six months (orders, a firmware fix, a support ticket, a refund, an
address change, a warranty question, a contact preference). All names and
facts are invented. Probes in `probes/sample_100.yaml` include bare identifiers
(order numbers, ticket numbers), a superseded-fact question (old vs new
address), and an abstention probe whose correct answer is UNKNOWN because the
fact was never stated.

## Why the reduction is higher than the private run

Reduction is a ratio. This corpus is short casual chat (about 2.7k tokens
pasted per question; recall contexts ran 71-370 tokens). The private corpus is
dense technical text (25k pasted, ~4k recalled). Each results file reports its
own corpus; across the two, the measured range is 82-92%.

## Disclosures

- Two probe questions were reworded during development, before this published
  run. Both originally asked about "Sam" by name; the corpus speaker is only
  ever "user" and the name Sam never appears in the conversation text, so the
  questions were unanswerable from the data as written. They now ask in first
  person ("What is my partner's name?"), which is how a user queries their own
  memory. Expected answers were not changed, and no grading, prompt, or
  retrieval setting was changed.
- Before the rewording, one of those probes surfaced a real ranking finding:
  for a "what did I decide" question, recall returned specification facts about
  the entity and ranked the decision fact at the score floor. Intent-aware
  fact ranking is on the product roadmap; the benchmark will be rerun after
  that work lands.
